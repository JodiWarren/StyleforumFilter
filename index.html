Styleforum Currency Filter.

This set of bookmarklets will allow for quick and easy browsing of the Styleforum marketplace.

Drag to your bookmarks bar

GBP EURO USD

Usage:

  • First click: hides all non-applicable currencies. If there are no matching items, go to the next page.
  • If there were matching items, a second click will advance to the next page.
